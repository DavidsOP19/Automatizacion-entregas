# Automatización de Reportes de Entregas Logísticas

## Descripción
Script Python que automatiza la generación de reportes de entregas para empresas de logística. Procesa datos de pedidos y genera un reporte Excel profesional con métricas, análisis por región y repartidor, y gráficas automáticas.

## Tecnologías
- Python 3.14
- Pandas 3.0.2
- OpenPyXL 3.1.5
- GitHub

## Resultados
- 50 pedidos procesados
- Tasa de entrega: 68%
- Tiempo promedio: 1.9 días
- Reporte con 4 hojas y gráfica de barras

## Cómo ejecutar
```bash
pip install pandas openpyxl
python report_generator.py
```
